## OpenScreen とは

**OpenScreen** は、プロダクトデモやウォークスルー動画を作成するための**無料・オープンソースのデスクトップアプリ**。画面録画 → 編集（ズーム・アノテーション・トリミング等） → MP4/GIF エクスポートを一気通貫でこなす。技術スタックは Electron + React + PixiJS（WebGL レンダリング）+ TypeScript。macOS / Windows / Linux 対応。

---

## 何が嬉しいのか — 既存ツールとの比較

| 観点 | **Screen Studio**（有料 $29/月） | **OBS + 外部エディタ** | **OpenScreen** |
|---|---|---|---|
| **コスト** | サブスクリプション必須 | 無料だが編集は別途 | **完全無料・MIT ライセンス** |
| **ウォーターマーク** | なし（有料版） | なし | **なし** |
| **商用利用** | ライセンス次第 | ○ | **○（制限なし）** |
| **録画→編集の一体感** | ◎ シームレス | ✕ ツール間の受け渡しが手間 | **○ 録画後そのまま編集画面へ** |
| **自動ズーム提案** | ○ | ✕ | **○ カーソル滞留パターンを解析し自動提案** |
| **プラットフォーム** | macOS のみ | 全 OS | **全 OS** |
| **プライバシー** | ローカル | ローカル | **完全ローカル処理・クラウド不要** |
| **高度な編集機能** | ◎（カラーグレーディング等） | ◎（外部エディタ依存） | △ 基本機能に集中 |

**要するに：** 「Screen Studio ほどの高度な編集は不要だが、OBS + 動画編集ソフトの組み合わせは面倒」というユーザーにとって、**録画から見栄えの良いデモ動画の完成までを無料でワンストップ**に実現できる点が最大の価値。

## 使い方の流れ

```
┌─────────────┐    ┌──────────────┐    ┌─────────────┐    ┌────────────┐
│  1. 起動     │ →  │ 2. 録画設定   │ →  │ 3. 録画      │ →  │ 4. 編集     │ → 5. エクスポート
│  HUD表示     │    │ ソース選択    │    │ 操作を実演   │    │ タイムライン │
└─────────────┘    └──────────────┘    └─────────────┘    └────────────┘
```

1. **起動** — アプリを立ち上げると、画面上にフローティングの HUD（小さなツールバー）が表示される
2. **録画設定** — HUD から録画ソース（全画面 / 特定ウィンドウ）を選択。必要に応じてマイク・システム音声・Web カメラをトグルで ON/OFF
3. **録画** — 録画ボタンを押してデモ操作を実演し、停止ボタンで終了（最大 4K/60fps 対応）
4. **編集** — 録画停止後、自動的にエディタが開く。主な編集操作：
   - **ズーム／パン** — タイムライン上にズーム区間をドラッグ追加（カーソル滞留を解析した自動提案機能あり）
   - **アノテーション** — テキスト・矢印・画像をキャンバスにドラッグ＆ドロップ
   - **トリミング／速度変更** — 不要部分のカット、区間ごとの再生速度調整
   - **見た目の調整** — 壁紙（18種プリセット）、角丸、影、パディング、Web カメラの PiP 配置
   - **アスペクト比** — 縦・横・正方形を切り替え可能
5. **エクスポート** — MP4 または GIF を選択し、解像度・品質を指定して書き出し。`.openscreen` 形式でプロジェクト保存も可能

> **補足：** 現在ベータ段階（v1.3.0）。キーボードショートカットのカスタマイズ、多言語対応（英語・中国語・スペイン語）、プロジェクトの保存・再開にも対応している。
